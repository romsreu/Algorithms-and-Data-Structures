Aplicación desarrollada en C++ que integra gestión de usuarios, juegos educativos y un procesador de texto para analizar un libro real. Utiliza archivos binarios y secuenciales, matrices, listas enlazadas y una arquitectura modular propia de Algoritmos y Estructuras de Datos.

El sistema inicia con una pantalla de bienvenida y un login que valida legajo y contraseña desde un archivo binario, mostrando el último acceso y controlando intentos fallidos. Luego habilita un menú principal desde donde se accede al resto de las funciones.

Incluye herramientas para practicar idiomas, como la visualización de alfabetos en varios idiomas (minúsculas, mayúsculas y caracteres acentuados), y un conjunto de juegos implementados desde cero:

- Ahorcado, con selección aleatoria de palabras, control de intentos y sistema de puntaje.

- Palabra cifrada, basado en cifrado César.

- Números en otro idioma, con nombre escrito y representación gráfica tipo display.

- Memoria, donde el usuario recuerda patrones y obtiene puntajes por aproximación.

El módulo de Procesamiento de Texto permite analizar un archivo de libro y obtener:

- Frecuencias de caracteres (con exportación a frecuencia.txt).

- Una matriz que cruza letras iniciales con longitudes de palabras.

- Medidas globales como longitud promedio, top de palabras más frecuentes por letra y porcentajes de vocales vs consonantes.

- Una lista enlazada dinámica ordenada por frecuencia, longitud y orden alfabético, desde la cual se generan archivos filtrados por letra inicial.
